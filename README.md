# Wallet

A premium, offline-first Android money tracking application with a futuristic "Liquid Glass" design and advanced AI insights.

## Features

- **Theme Engine**: Expressive Material 3 & Liquid Glass aesthetics.
- **Core Data**: Robust local database with Isar.
- **Dashboard**: Interactive charts and summaries.
- **Advanced Finance**: Budgets, loans, bill splitting, goals.
- **AI Integration**: Local Gemini AI for spending analysis and forecasting.
- **Performance**: Optimized for 60fps+ with smooth animations.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- Flutter SDK
- Android Studio / VS Code

### Installation

1. Clone the repo
2. Run `flutter pub get`
3. Run `flutter run`

## Architecture

This project follows a clean architecture with Riverpod for state management.

- `lib/core`: Constants, theme, utils.
- `lib/features`: Feature-based modules.
- `lib/shared`: Shared widgets and providers.
